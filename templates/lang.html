Python

Python was created by Guido van Rossum and officially released in 1991. It is both a dynamic and imperative language. Python is notably known to stress the importance of code readability. It's simplicity in syntax relies heavily on the proper parsing pn whitespace rather than semicolons like Java.

There are several notable applications and companies that implement Python. Instagram utilizes Python 3 with the combination of the Django framework. Spotify takes Python and incororates it into its back-end services. Amazon, on the other hand, implements Python by analyzing customer's purchasing patterns as well as their search patterns. This is all in due to their sophisticated Python machine learning engine that analyzes their databases. Python is a great language and veratile in many aspects!

Racket

Racket was formerly known as "PLT Scheme". It is both a functional and static language. Racket is actually based off the Scheme language, which happens to be a dialect of Lisp. It was created by Matthias Felleisen in the mid 1990s. It was originally a project for his research group, and then expanded to be dedicated to produce teaching materials for beginner programmers.

Although Racket is not as popular as Python, it still has been implemented in varying real-life applications. For example, Racket has been implemented in the gaming industry. Specifically, a notable company that has utilizies Racket as their scripting language is Naughty Dog. Additionally, although not Racket, HackerNews written using Arc programming language, which is also a dialect of Lisp. Overall, Racket is a popular language used for reseaching and educational purpose.

Haskell

Haskell was not created by a single person; it was published through a committe that involved several researchers that included Simon Marlow, Simon Jones, and Phil Wadler among the many of them. A very notable feature about Haskell is that it can be interpreted (GHCI) or compiled (GHC).

There are various applications that use Haskell. Facebook utilizes Haskell is used for battling spam. ABN ARMO, which is a bank located in Amsterdam, incorporates Haskell in a varying range of company projects. Barclays, a bank as well, also incorporates Haskell into their code base.
