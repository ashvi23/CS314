Data Structures



Data structures are utilized in every single language. Although the general Big O of each type of data structure is fairly similar over different types of languages, the way the structures are implemented differ. Each language has different features that might make a particular data structure harder or easier to implement. In this project, we want to explore the relationship between languages of different paradigms and the data structures.


Stacks:
Stacks are a basic data structure that are often times some of the first things programmers learn. We wanted to start of by implementing this using racket, haskell, and python.

Queue:
Queues are another data structures we chose to implement. This structure was interesting because it worked on both sides of a "list". We wanted to know how haskell and racket, two languages that work with lists, compare in terms of Big O to python. Would dequeue still require O(n)? Is there a faster way to implement queues using haskell and racket.
