Dynamic

Dynamic langauges are languages where every declared variable is bounded to an object, unless the variable is null. Most languages that are dynamic are also dyanmically typed. Meaning, during execution time, variable names will be assigned to objects. There is also a possibility that a variable is bounded to different types during execution. The main idea is that the type of a variable is only known when the programming is running and not beforehand. The advantages of dynamic languages are that they are much more flexible unlike static languages, however, it makes room for more unexpected errors because the typing is not checked before compiling but rather on the fly.
Languages: Python, Racket, R, Javascript, Erlang and more!
